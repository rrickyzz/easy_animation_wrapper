Intuitive API for easy animation.

## Getting started

```

flutter pub add easy_animation_wrapper

```

## Usage

```dart
  @override
  Widget build(BuildContext context) {
    return const Center(
      child: SlideVerticalAnimation( child:  Text('Slide Vertical Animation')),
    );
  }
```
